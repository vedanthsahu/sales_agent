# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1qKMWuVGeC0cFkmU0isnDajRMVCty910C

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure backend environment in `sales-assist-backend/.env` (LLM keys live only on the backend).
3. Run the app:
   `npm run dev`
